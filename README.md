# NeteaseMusic Unlocker

解锁网易云音乐客户端变灰歌曲 修改 by NULL QQ:729893406 & 小宇 QQ:2083664136

## 特性

- 使用其他软件音源替换无版权歌曲链接
- 根据nondanee的UnblockNeteaseMusic项目更改而来，做了优化和一键脚本
- 无需手动进行繁琐的配置，运行即用，自动配置

## 运行

1. 下载最新的版本
2. 解压文件
3. 运行NeteaseMusic Unlocker.bat
4. 完成
